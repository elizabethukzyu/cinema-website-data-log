# cinema-website-data-log
本仓库定位为影视网站信息记录日志，用于长期保存互联网上可访问的影视类站点基础资料。整理内容包括站点名称、域名地址、内容分类及访问状态说明等，不涉及站点内部具体内容。仓库不存储影视资源，不提供播放、下载或跳转指引，所有记录均基于公开资料整理完成。本项目仅用于资料留存与信息索引，不构成任何形式的推荐、背书或商业用途。
